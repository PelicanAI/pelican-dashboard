# Market Section Icons

Place the following PNG icon files in this directory:

- **us-stocks.png** - Metallic US flag icon for US Stocks section
- **crypto.png** - Metallic crypto coin icon for Crypto section  
- **forex.png** - Metallic currency symbols icon for Forex section

## Icon Specifications
- Height: 32px (h-8) with auto width to maintain aspect ratio
- Style: Metallic purple with built-in glow effect
- Hover effect: Additional purple drop-shadow glow

The icons are referenced in `src/components/market-stats/MarketStatsSection.tsx`
